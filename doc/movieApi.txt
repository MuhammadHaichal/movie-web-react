BASE URL MOVIES LIST
===================

Ada kategori =

- Top Rated  ( state default pakai yang Top Rated yang di movies list )
- Popular
- Upcoming
- Now Playing


# Top Rated
===========
Get a list of movies ordered by rating.

base url = https://api.themoviedb.org/3/movie/top_rated?language=en-US&page=1




# Popular
=========
Get a list of movies ordered by popularity.

base url = https://api.themoviedb.org/3/movie/popular?language=en-US&page=1




# Upcoming
==========
Get a list of movies that are being released soon.

base url = https://api.themoviedb.org/3/movie/upcoming?language=en-US&page=1



# Now playing
=============
Get a list of movies that are currently in theatres.

base url = https://api.themoviedb.org/3/movie/now_playing?language=en-US&page=1









DETAIL MOVIE LIST
================
Get the top level details of a movie by ID.

base url = https://api.themoviedb.org/3/movie/movie_id?language=en-US
